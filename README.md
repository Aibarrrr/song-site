<h1>Арадағы үнсіздік</h1>

<p>
Күндер өтеді үнсіз ғана,
  
Ойлаймын сені тағы да.
  
Жауап күттім,бірақ сенен
  
Суық үн келді арада.

Жүрек сен деп соғады әлі,

Бірақ шаршадым күтіп мен,

Жақын болдың бір сәт қана,

Сосын алыстадың бірден.

Көзім талды жол қараумен,

Үміт үзбей тағы да,

Бір хабарға зәру болып,

Қалдым үнсіз тағдырға.

Сенсіз кештер ұзақ әрі,

Ойларым да шашырап,

Бір үміт пен бір күдікпен

Жүрмін өзім адасып.

Бәлкім,бәрі өткінші ме,

Сезім жалган,уақыт па?

Жауап таппай жүрегімде

Қалды белгі,сұрақ та.



</p>
